For Administrators: How to Update Events

To add, modify, or remove events, edit the EVENTS array in the js/data.js file.

Each event requires:

  • id: Unique identifier number
  • title: Event name (e.g., "Annual Meeting 2024")
  • date: Date in YYYY-MM-DD format (e.g., "2024-03-15")
  • time: Event time (e.g., "10:00 AM")
  • venue: Location of the event
  • description: Brief description of the event
  • status: "upcoming" or "completed"
